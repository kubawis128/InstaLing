# InstaLing

InstaLing Bot

# Launching
Open command line in parent directory
### Linux:
```
$ python3 main.py
```
### Windows
```
py main.py
```
# Requierments

``` tktinter ```

``` python3 + pip```

``` selenium ```

``` googletrans ``` == 4.0.0-rc1

``` ttkthemes ```

``` geckodriver ```

# Installing

On Arch linux

```
$ sudo pacman -Sy tk python-pip python geckodriver
$ sudo pip3 install tk selenium googletrans==4.0.0-rc1 ttkthemes
```
